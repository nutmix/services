# Skycoin Tester CLI Utility



## supported platforms
1. Windows
1. OS X
1. Linux

## pre-requisits

1. go installed & GOPATH set.
1. skycoin installed
```bash
$ go get github.com/skycoin/skycoin
```
1. skycoin running locally

## Installation

```bash
$ cd $GOPATH/src/github.com/skycoin/services/load
$ go install 
```

## Usage

```bash
$ cd $GOPATH/src/github.com/skycoin/services/load
$ go run load.go
```

or 
```bash
$ $GOPATH/load.exe
```
